# ADR-003: Session Management Design

Status: accepted

## Context
Users need a way to manage their active sessions, including viewing and revoking them.

## Decision
We will create a dedicated session management page that lists active sessions and allows users to revoke them.

## Consequences
- Users can manage their sessions easily.
- Security is enhanced by allowing users to revoke suspicious sessions.
