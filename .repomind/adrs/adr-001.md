# ADR-001: OAuth Provider Selection

Status: accepted

## Context
We need to support multiple OAuth providers to allow users to log in using their existing accounts.

## Decision
We will implement OAuth login using Google and GitHub as they are widely used and provide robust authentication services.

## Consequences
- Users can log in using their Google or GitHub accounts.
- Future providers can be added following a similar pattern.
